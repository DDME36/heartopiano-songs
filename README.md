# 🎹 HEARTOPIANO

Auto Piano Player for Heartopia Game

---

## ✨ Features

- 🎵 Online song library with auto-updates
- ⚡ Speed control (0.5x - 2.0x)
- 🎨 Beautiful glassmorphism UI
- 🎹 Mini mode for compact playing
- 🔍 Search & filter songs
- 🌐 Multi-language (TH/EN)
- ⌨️ Keyboard shortcuts
- ✅ No installation required - Just run!

---

## 📋 Requirements

- Windows 10/11
- Internet connection

**Note:** Python is included in the app!

---

## 🎮 How to Use

1. Open Heartopia game
2. Run Heartopiano
3. Select a song
4. Adjust speed
5. Press F5 or Play button
6. Enjoy!

### Keyboard Shortcuts

- `F5` - Play/Pause
- `F8` - Stop
- `Space` - Play/Pause
- `Esc` - Stop
- `Ctrl + ↑` - Increase speed
- `Ctrl + ↓` - Decrease speed
